# Cafe Sales Data Cleaning & Analysis Project
## Project Overview
This project focuses on cleaning and analyzing a cafe sales dataset for the year 2023. The goal was to transform raw, inconsistent data into a structured dataset and generate meaningful business insights through analysis and visualization.
## Data Cleaning Process
The raw dataset contained inconsistencies such as duplicates, missing values, and invalid entries.
The following cleaning steps were performed in Excel:
- Removed duplicate records to ensure data integrity  
- Filtered and eliminated rows containing:
  - Errors  
  - Blank values  
  - "Unknown" entries  
- Ensured consistency across all columns  
## Dataset Description
The dataset includes the following fields:
- Transaction ID  
- Item  
- Quantity  
- Price per Unit  
- Total Spent  
- Payment Method  
- Location (In-store / Takeaway)  
- Transaction Date  
## Data Analysis
After cleaning, pivot tables were created to explore key business metrics:
- Number of Transactions per Month  
- Revenue per Item  
- Quantity Sold per Item  
- Revenue per Month  
These analyses helped uncover sales trends, top-performing products, and customer purchasing patterns.
## Dashboard
An interactive dashboard was built in Excel to visualize insights.
### Features:
- Dynamic filtering using slicers:
  - Location (In-store / Takeaway)  
  - Payment Method  
- Clear visualization of:
  - Monthly revenue trends  
  - Product performance  
  - Sales distribution  
## Key Insights 
- Peak sales months and seasonal trends  
- Best-selling items by revenue and quantity  
- Preferred payment methods  
- Differences between in-store and takeaway sales
